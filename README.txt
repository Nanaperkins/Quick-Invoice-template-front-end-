What is new
-----------
- Animated add/remove rows and totals pulse.
- Live currency converter with 12 hour cache (uses exchangerate.host). Graceful fallback if offline.
- Still zero backend. Perfect for GitHub Pages or Netlify.

How to run
----------
1. Extract the zip.
2. Open index.html.
3. Add items, watch totals and the conversion widget update.
4. Print to PDF, save or load templates, export or import JSON.

Deploy
------
- GitHub Pages: push and enable Pages (entry file index.html).
- Netlify: drag and drop the folder.

Notes
-----
- Live rates require internet access in the browser. If the fetch fails, the widget falls back to base currency totals.
- You can change the base invoice currency and the display currency independently.
